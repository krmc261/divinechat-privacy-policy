1. Overview

DivineChat is a mobile application intended to provide educational and faith-based conversational experiences with simulated biblical figures. This privacy policy outlines how we handle any data processed or encountered during use of the application.

2. Information We Do Not Collect

DivineChat does not collect or store personally identifiable information (PII). We do not ask for your name, email, address, phone number, or other personal identifiers. We do not store any user messages or behavioral tracking data on our servers.

3. How We Use Your Input

User input is temporarily sent to OpenAI’s API in order to generate a conversational response. These interactions are not stored by DivineChat. OpenAI may retain data temporarily to monitor for abuse or to improve model performance. For full details, see OpenAI’s Privacy Policy.

DivineChat uses OpenAI’s API to generate responses. We do not use advertising SDKs, analytics platforms, or third-party tracking tools. No third-party has access to user messages outside the temporary API processing described above.

6. Children’s Privacy

DivineChat is not intended for use by children under the age of 13. We do not knowingly collect data from anyone under this age. If a parent or guardian believes a child has submitted personal information, they should contact us immediately to ensure data is removed.
